## 📦 Installation

### Prerequisites

- Linux desktop environment
- Node.js and `pnpm` (`pnpm@10.13.1` is declared in the workspace)
- Rust toolchain
- Tauri Linux system dependencies from the [official prerequisites guide](https://v2.tauri.app/start/prerequisites/)

From the repository root:

```bash
pnpm install
pnpm tauri:dev

# build
pnpm tauri:build
```

Other Tauri CLI subcommands from root use argument forwarding: `pnpm tauri -- info`. For X11-specific dev (matches `apps/desktop` `tauri:dev:x11`): `pnpm tauri:dev:x11`. Vite-only: `pnpm desktop:dev`, `pnpm desktop:build`, `pnpm desktop:preview`.
